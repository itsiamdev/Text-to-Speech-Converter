# Text to Speech Converter

A simple and intuitive web-based application that converts text input into speech output using the browser's built-in Speech Synthesis API. This tool allows users to enter text and listen to it being read aloud with customizable voice options.

## Features

- **Text Input**: Enter any text in the textarea to convert it to speech.
- **Voice Selection**: Choose from available system voices to customize the speech output.
- **Responsive Design**: Clean, modern UI with a gradient background and responsive layout.
- **Easy to Use**: Single-click button to start listening to the text.

## Technologies Used

- **HTML5**: Structure and layout of the application.
- **CSS3**: Styling with modern features like flexbox and gradients.
- **JavaScript**: Handles speech synthesis and voice selection using the Web Speech API.
- **Web Speech API**: Browser-native API for text-to-speech functionality.

## Usage

1. Open the application in a modern web browser that supports the Web Speech API (e.g., Chrome, Firefox, Safari).
2. Enter your desired text in the textarea.
3. Select a voice from the dropdown menu (if available).
4. Click the "Listen" button to hear the text being spoken.
